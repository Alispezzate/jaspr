# Style
*A component that renders a \<style\> element.*

---

To define a global style element with a set of style rules use the `Style` (without *s*) component in combination with
the `css()` method:

```dart
Style(styles: [
    css('.red-text').text(color: Colors.red),
    css('#content', [
      css('&').box(width: 100.percent, height: 100.percent),
      css('a').text(color: Colors.green),
    ]),
]);
```

will render as:

```html
<style>
  #content {
    width: 100%;
    height: 100%;
  }

  #content a {
    color: green;
  }

  .red-text {
    color: red;
  }
</style>
```
